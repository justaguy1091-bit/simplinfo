# Manual Installation

## Step 1
Create the file in your home directory:

[text editor] simplinfo

Examples:
ee simplinfo
nano simplinfo
vi simplinfo

## Step 2
Paste the Simplinfo (or “Simplinfo LE” if you’re on Linux) source code and save the file.

## Step 3
Make it executable:

chmod +x simplinfo

## Step 4
Run Simplinfo:

./simplinfo

## Step 5
Enjoy!
