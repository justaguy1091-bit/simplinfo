# Simplinfo Installation

## Step 1
Create a directory for the project:

mkdir terminfo

## Step 2
Enter the directory:

cd terminfo

## Step 3
Create the script:

sudo [text editor] simplinfo

Examples:
sudo ee simplinfo
sudo nano simplinfo
sudo vi simplinfo

## Step 4
Paste the Simplinfo (or “Simplinfo LE” if you’re on Linux) source code and save the file.

## Step 5
Make it executable:

chmod +x simplinfo

## Step 6
Run Simplinfo:

./simplinfo

## Step 7
Enjoy!
